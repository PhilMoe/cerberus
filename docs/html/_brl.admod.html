Detailed Discussion

The admob module provides simple support for the Google Admob advertising system, which allows you to add advertising to your projects.

Admob ads are only available on the extended ios and android targets.

To add Admob advertising to your project:

  • 1 : Create an Admob account and add a new site/app for your project. You can create an Admob account at http://www.admob.com

  • 2 : Add the app config setting ADMOB_PUBLISHER_ID to your project. This must be set to your app's Admob 'Publisher ID' - eg: 'a1507f4c28c6f2a'. See App config settings.

  • 3 : Import the brl.admob module into your project and use the EnableAds function to enable ads.

    • Example

    'Make sure to select an '(Extended)' target!
'
#ADMOB_PUBLISHER_ID="a1507f4c28c6f2a"

    
Import mojo
Import brl.admob

    
Class MyApp Extends App

    
	Field enabled:=False
	Field layout:=6
	
	Method OnCreate()
		SetUpdateRate 60
	End
	
	Method OnUpdate()
		If MouseHit( 0 )
			If enabled
				DisableAds
				enabled=False
			Else
				layout+=1
				If layout=7 layout=1
				EnableAds 1,layout
				enabled=True
			Endif
		End
	End
	
	Method OnRender()
		Cls
		DrawText "Click to toggle ads!",DeviceWidth/2,DeviceHeight/2,.5,.5
	End
	
End

    
Function Main()
	New MyApp
End

    Function AdsAvail : Bool ()

    Function EnableAds : Void ( style:Int=1, layout:Int=2 )

    Enables ads. The style and layout parameters should be one of the following:

    1320x50 Banner ad

    1top left
    2top center
    3top right
    4bottom left
    5bottom center
    6bottom right
